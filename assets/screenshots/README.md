# Screenshots / Capturas de Tela

Este diretório contém capturas de tela dos sistemas do portfólio SirTechSolutions.

## Como adicionar capturas de tela

1. Salve as imagens neste diretório (`assets/screenshots/`)
2. Use nomes descritivos, por exemplo:
   - `barbersys-dashboard.png`
   - `barbersys-agendamentos.png`
   - `sir-petshop-dashboard.png`
   - `gcm-sistema-dashboard.png`
   - `sentinel-condo-dashboard.png`
3. Referencie as imagens nas páginas de projeto em `docs/projects/`

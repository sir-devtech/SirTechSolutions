# Assets de Imagens

Esta pasta armazena screenshots e imagens demonstrativas utilizadas no portfólio público do SirTechSolutions.

## Política de uso de imagens

Antes de adicionar qualquer imagem, revisar se ela **não expõe**:

- dados pessoais ou sensíveis de usuários
- chaves, tokens ou credenciais
- rotas, URLs internas ou caminhos do sistema
- procedimentos operacionais
- documentos confidenciais
- detalhes técnicos críticos de arquitetura
- informações estratégicas de negócio

## Organização sugerida

```
assets/images/
├── barbersys/          # Screenshots do BarberSys
├── gcm-sistema/        # Screenshots do GCM Sistema
├── sir-petshop/        # Screenshots do SirPetShop
└── sentinel-condo/     # Screenshots do Sentinel Condo
```

## Observação

O objetivo das imagens é demonstrar qualidade visual, capacidade de construção e contexto dos sistemas — sem comprometer a segurança, confidencialidade ou privacidade das soluções apresentadas.

Toda imagem deve ser revisada antes da publicação.

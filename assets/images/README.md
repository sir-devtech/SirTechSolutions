# Assets de Imagens

Esta pasta pode armazenar screenshots e imagens demonstrativas usadas no portfólio público.

## Recomendação de uso

Antes de adicionar qualquer imagem, revisar se ela **não expõe**:

- dados pessoais sensíveis
- chaves, tokens ou credenciais
- rotas, URLs e caminhos internos
- procedimentos operacionais
- documentos confidenciais
- detalhes técnicos críticos

## Sugestão de organização

```
assets/images/
├── barbersys/
├── gcm-sistema/
├── sir-petshop/
└── sentinel-condo/
```

## Observação

O objetivo das imagens é demonstrar qualidade visual, capacidade de construção e contexto dos sistemas, sem comprometer a segurança das soluções.

# SirTechSolutions

Portfólio público de soluções desenvolvidas por **Moisés / sir-devtech**, desenvolvedor **full-stack** com experiência prática em ambiente profissional — abrangendo sistemas administrativos, aplicações web, PWAs e soluções multiplataforma.

> **Aviso de confidencialidade:**
> Este repositório tem finalidade **profissional e demonstrativa**.
> Os sistemas reais apresentados aqui possuem implementação **privada e confidencial**.
> Por esse motivo, **não são publicados** neste repositório:
>
> - código-fonte dos sistemas operacionais
> - rotas, endpoints e caminhos internos
> - estrutura de banco de dados
> - documentação técnica privada
> - guias internos e procedimentos operacionais
> - configurações de infraestrutura e deploy
> - critérios de segurança e detalhes sensíveis de negócio

## Perfil profissional

Desenvolvedor full-stack com atuação em projetos reais, do planejamento ao deploy em produção. Experiência prática com:

- **Arquitetura e Clean Code** — organização modular, separação de responsabilidades, código legível e sustentável

- **Monitoramento e observabilidade** — uso de ferramentas como **Sentry** para rastreamento de erros em produção
- **Automação de processos** — scripts, integrações e fluxos automatizados para redução de trabalho manual
- **Sistemas multiplataforma** — soluções web, PWA instalável e apps Android nativos dentro do mesmo ecossistema

### BarberSys

Sistema de gestão para serviços de atendimento e rotinas operacionais, desenvolvido em **TypeScript** com **Progressive Web App (PWA) configurada e funcional** — instalável em dispositivos mobile e desktop diretamente pelo navegador, sem necessidade de loja de aplicativos.

**Destaques:**
- sistema web responsivo com suporte a instalação via PWA
- gestão de cadastros, atendimentos e rotinas administrativas
- organização modular e código estruturado com foco em manutenibilidade
- interface prática para uso operacional diário

**Stack (nível seguro):** TypeScript · HTML · CSS · PWA

[Ver detalhes do projeto →](docs/projects/barbersys.md)

---

### GCM Sistema

Plataforma operacional e administrativa com presença **multiplataforma**: aplicação web, **PWA instalável** e **app Android nativo**, entregando experiência consistente em todos os dispositivos.

**Destaques:**
- dashboard operacional com visão consolidada
- módulos administrativos organizados por área funcional
- aplicativo Android integrado ao ecossistema do sistema
- PWA disponível para instalação mobile/desktop
- foco em produtividade e centralização de informações institucionais

**Stack (nível seguro):** Django · Python · HTML · Kotlin · Shell · PWA · Android

[Ver detalhes do projeto →](docs/projects/gcm-sistema.md)

### SirPetShop

Sistema de gestão para pet shops com foco em centralização de rotinas administrativas. Base estruturada em **Python/HTML/Shell**, com evolução contínua planejada para ampliar funcionalidades e cobertura de testes.

**Destaques:**
- operação administrativa centralizada
- gestão de cadastros e atendimentos
- base sólida com potencial de expansão modular
- projeto em evolução ativa

**Stack (nível seguro):** Python · HTML · Shell

[Ver detalhes do projeto →](docs/projects/sir-petshop.md)

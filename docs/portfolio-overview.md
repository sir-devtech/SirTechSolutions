## 🔒 Nota de Confidencialidade

Todos os sistemas listados aqui são **proprietários e privados**. Código-fonte, rotas, configurações, documentação interna e procedimentos operacionais não são divulgados publicamente. Este portfólio apresenta apenas informações seguras de produto.

---

## 🗂️ Sistemas

### [🔵 BarberSys](./projects/barbersys.md)
Sistema completo de gestão para barbearias. Controla agendamentos, cadastro de clientes, serviços, profissionais e financeiro.

### [🟣 SirPetShop](./projects/sir-petshop.md)
Sistema de gestão para pet shops. Gerencia vendas, estoque, atendimentos, cadastro de pets e tutores.

### [🟢 GCM Sistema](./projects/gcm-sistema.md)
Sistema de gestão comercial multiuso. Controla clientes, produtos, serviços, pedidos e relatórios gerenciais.

### [🟠 Sentinel Condo](./projects/sentinel-condo.md)
Sistema de controle de acesso para condomínios. Gerencia moradores, visitantes, porteiros e registros de entrada/saída.

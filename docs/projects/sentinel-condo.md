# Sentinel Condo

## Visão geral

O **Sentinel Condo** é um projeto voltado para gestão condominial, com foco em centralização administrativa, monitoramento, organização operacional e visão consolidada do ambiente.

## Problema que o sistema resolve

A administração condominial envolve múltiplas demandas: controle, acompanhamento, perfis de acesso, monitoramento, registros e organização de informações.

O Sentinel Condo foi pensado para oferecer uma experiência administrativa mais unificada, moderna e funcional.

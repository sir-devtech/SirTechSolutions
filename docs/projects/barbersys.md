# BarberSys

## Visão geral

O **BarberSys** é uma solução voltada para gestão de operações e rotinas de atendimento no contexto de serviços, desenvolvida em **TypeScript** com **Progressive Web App (PWA) configurada e funcional**.

O sistema pode ser instalado diretamente pelo navegador em dispositivos mobile e desktop, oferecendo experiência próxima à de um aplicativo nativo — sem necessidade de publicação em loja de aplicativos.

## Estágio do projeto

> **✅ Em produção** — sistema ativo, com PWA pronta e funcional.

## Problema que o sistema resolve

Negócios de atendimento e prestação de serviços frequentemente sofrem com processos descentralizados, controle manual e baixa visibilidade operacional.

O BarberSys foi pensado para apoiar esse cenário com uma estrutura mais organizada, digital e acessível de qualquer dispositivo.

## Principais funcionalidades

> Esta lista é apresentada em nível alto, sem expor detalhes internos da implementação.

- gestão de cadastros e rotinas de atendimento
- centralização de informações operacionais
- apoio à administração do negócio
- melhoria da visibilidade gerencial
- suporte a fluxo administrativo integrado
- instalação via PWA para uso mobile e desktop

## Público-alvo / perfis de usuário

- administrador do estabelecimento

## Tecnologias / stack em nível seguro

- **TypeScript** — linguagem principal do sistema
- HTML · CSS — estrutura e estilo da interface
- **PWA** — instalação offline, service worker configurado e funcional
- Organização modular e boas práticas de Clean Code

## Diferenciais

- **PWA pronta e funcional** — instalável em qualquer dispositivo
- TypeScript para maior segurança de tipos e manutenibilidade
- foco em praticidade e uso operacional diário
- organização modular com código limpo e sustentável
- base com potencial de expansão funcional contínua

# GCM Sistema

## Visão geral

O **GCM Sistema** é uma plataforma operacional e administrativa **multiplataforma**, disponível como aplicação web, **PWA instalável** e **aplicativo Android nativo** — garantindo experiência consistente independentemente do dispositivo do usuário.

O sistema centraliza informações, módulos administrativos e ferramentas de acompanhamento em um único ambiente de gestão.

## Estágio do projeto

> **✅ Em produção** — sistema ativo com versão web, PWA funcional e app Android disponível.

## Problema que o sistema resolve

Ambientes operacionais com múltiplas frentes de atuação exigem clareza, painel consolidado e fácil acesso às informações importantes — em qualquer dispositivo.

O GCM Sistema foi desenvolvido para oferecer esse acompanhamento por meio de uma experiência unificada, acessível via navegador, instalável como PWA ou utilizável como app Android nativo.

## Principais funcionalidades

> Descrição em nível alto, sem expor elementos internos da solução.

- dashboard operacional com visão consolidada
